# 競合調査と配布戦略 (2026-07-18)

deep-research ワークフロー(109エージェント・3票反証投票)+追加調査2本の結果。
前提: 開発者はAPIコストを負担しない(ユーザーのClaudeサブスク/BYOK)。無料公開可。
ローカル完結は「楽にやりたい」が動機であり、思想的コミットではない。

### 学術的裏付け

arXiv 2509.16811 (2025-09, n=15): エージェント型編集は長尺課題で Descript/OpusClip を有意に上回る
(p<.001)。主要不満は**処理遅延・プレビュー欠如・途中介入不能**。
→ vedit のライブプレビュー+可逆revisionログはこの弱点への直接対抗設計。

## 2. veditの空きポジション(調査の結論)

- 「会話型AI編集」自体はもう空いていない。Descript・Adobe・Riverside が製品化済み、カテゴリは確立段階。
- 空いているのは組み合わせ: **ローカル完結 × 会話型 × 完全可逆ログ × プロNLE引き渡し(OTIO+SRT)**。
  この4点を兼ねる製品は本調査で確認されなかった(悉皆保証はない)。
- 「買い切り+完全ローカル」は Recut/Timebolt/Resolve のみで、いずれも会話型なし。
  「会話型×ローカル」は空白地帯。
- 最大の脅威は Adobe AI Assistant の機能拡大速度(公開1か月で初稿領域に到達)と、
  Resolve の IntelliScript(ローカル+台本→タイムライン)。時間窓は狭いと見るべき。

## 3. 配布チャネル

- **Anthropic公式プラグインディレクトリ**(claude-plugins-official 約32.3k stars)に個人提出ルートあり
  (platform.claude.com/plugins/submit)。条件: 公開GitHubリポジトリ必須、課金機構なし=無料配布前提。
  → veditの前提(無料・BYOK不要=ユーザーのClaudeサブスクで動く)と完全に整合。**最有力チャネル**。
- コミュニティマーケットプレイス(anthropics/claude-plugins-community)も並存。
- Mac App Store 案: BYOK審査実態・whisper同梱可否は今回検証を通過した情報なし(未回答)。
  ただしAPIコスト非負担+無料でよい、という条件下では調べる必要性自体が低い。
- **ライセンス注意**: 同梱/依存ffmpegが libx264 入り(--enable-gpl)だと全体にGPL v2が及ぶ。
  配布前にビルド構成の確認が必要(ffmpeg.org/legal.html)。ユーザー側でbrew install ffmpeg させる
  「依存として要求」方式なら同梱問題は回避できる。

## 4. 無料公開のリターン実例(2024-2026)

| パターン | 実例 | 得たもの |
|---|---|---|
| 企業スポンサー | OpenCut(OSS CapCut代替、〜48k stars) | Vercel/fal.aiスポンサー。有料化せず |
| 別チャネル有料版 | Buzz(whisperオフライン文字起こし) | GitHub無料OSS+Mac App Storeで有料版の二本立て |
| 法人ライセンス | Remotion | 個人無料+4人以上企業に$25/dev/月。Claude用スキル公開でバイラル化 |
| 評判→大手入社 | Peter Steinberger(VibeTunnel→OpenClaw 20万stars) | OpenAI参画 |
| デファクト地位 | Superpowers(Claude Codeプラグイン、〜22万stars) | 公式マーケットプレイス採択。収入はほぼゼロ |

要点: **スター数と収入は比例しない**。無料公開の現実的リターンは評判・機会・スポンサーであり、
後から収益化するなら Buzz 型(OSS+ストア有料版)か Remotion 型(法人課金)が先例。

## 5. 推奨(この調査からの結論)

1. **Claudeプラグイン/スキルとして無料公開、公式ディレクトリへ提出**が第一手。
   APIコスト非負担・無料可・公開リポジトリ可の条件をすべて満たす唯一のチャネル。
2. 訴求軸は「会話型」ではなく「**素材が一切外に出ない会話型編集+完全Undo+Resolveへ引き渡せる**」。
   競合が全部クラウドである事実が最強のコピーになる。
3. Mac App Store は追わない(条件的に不要、審査調査も未完)。
   後日収益化したくなったら Buzz 型二本立てが先例。
4. 公開前の実務: ffmpegのライセンス確認(依存要求方式を推奨)、README英語化、
   他環境動作の検証(HANDOFFの「実行検証だけが捕まえる」原則は他人のMacでも同じ)。
5. 急ぐ理由: Adobe AI Assistant がベータ1か月で初稿領域に到達しており、
   「ローカル×会話型」の空白が空いている時間窓は長くない。

## 調査の限界

- 「ローカル完結の会話型競合が無い」は本調査範囲での不在(消極的証拠)。
- 商用製品の能力記述は主にベンダーコピー、編集品質は未検証。
- Mac App Store のBYOK/whisper同梱の審査実態は未回答のまま。
- 一部価格は情報源間で不一致(Recut $99/$129、Resolve無料版のAI機能線引き)。
- 時点依存が強い(Adobe急拡大中、プラグイン課金方針も変わりうる)。2026-07-18時点。
