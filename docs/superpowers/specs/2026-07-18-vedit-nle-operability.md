# 波E: NLE 操作性パック — 普通に使える直接操作

日付: 2026-07-18。動機(ユーザー): 「UIデザイン普通のNLEみたいに色々
使いやすいようにしたい」。これは製品の賭けの**改訂**である: 旧
「UI に razor を作らない」等の操作制限を撤回し、**既にタイムラインに
在るものへの操作**は商用 NLE 同等に完備する。会話側に残るのは
「構造の生成」(素材からの選定・構成・演出判断・書き出し実行)と
面倒作業の代行。アプリ内チャット入力を置かない原則は不変。

## E-1. コア ops(src 先行 — N6 オーバーレイ着地後)

- `splitClip(m, clipId, tlTime)`: クリップをタイムライン時刻で2分割
  (内容不変・境界追加のみ。字幕 cue/アンカーの追従は既存 Segment 機構で
  自然に成立することをテストで固定)。
- `duplicateClip(m, clipId)`: 直後に同一ソース範囲を複製配置。
- 論理 undo/redo スタック: 現行 restore ベースを UI から連打できる形に
  (daemon に `POST /api/undo` `/api/redo` — revision 履歴上のポインタ移動を
  restore op で表現。redo は undo 直後のみ有効、通常編集で破棄)。
- CLI: `vedit split <clipId> --at <tl秒>` / `vedit duplicate <clipId>` /
  `vedit undo` / `vedit redo`(--base 流儀)。

## E-2. キーボード(web)

Space=再生/停止、J/K/L=逆送り(将来)/停止/早送り(1x/2x トグル)、
←/→=1フレーム、Shift+←/→=1秒、Home/End=先頭/末尾、
I/O=範囲イン/アウト(ループ再生用)、S=プレイヘッドで分割、
Delete/Backspace=選択削除(確認なし・undo で戻す)、
Cmd/Ctrl+Z=undo、Shift+Cmd+Z=redo、+/-=ズーム。
ヘルプ「?」オーバーレイに一覧を追記。入力中(contentEditable/フォーム)は
すべて無効。

## E-3. タイムライン直接操作(web)

- **分割**: S キー+クリップ右クリック「ここで分割」+計器列の分割ボタン
  (プレイヘッド位置)。
- **右クリックメニュー**(クリップ/テロップ cue/オーバーレイ/BGM):
  分割・複製・削除・ミュート(音量シートへ)・プロパティ(ドリルインシート)。
  既存のキーボード到達可能性と等価な操作のみ(メニュー限定機能を作らない)。
- **複数選択**: Shift+クリック/ドラッグ矩形。一括削除・一括移動。
- **ズーム/横スクロール**: 拡大スライダーを実機能化(現状を確認し、
  スタブなら実装)。+/- キー、trackpad ピンチ、Shift+ホイール横スクロール。
  ズーム中もプレイヘッド追従(再生中は auto-follow、手動スクロールで解除)。
- **範囲ループ**: I/O で範囲を設定→ループ再生(知覚確認用)。範囲の
  下見レンダー実行は会話側のまま(実行系は UI に置かない — 不変原則)。
- ドラッグ移動・端トリムは既存実装を維持・強化(スナップ、±1F 微調整)。

## E-4. undo/redo の UI

ヘッダーに戻す/やり直すボタン(disabled 状態を正直に)+ショートカット。
履歴タブとの整合(undo も履歴に載る現行の透明性を維持)。

## 変えないもの

- 書き出し実行ボタンは置かない(結果ビューのみ — 従来どおり)。
- compose(紙芝居の構成)ビルダー UI は作らない(会話側)。
- アプリ内チャット入力を作らない。
- 「Cowork に頼む」チップ導線は維持(手でもできるが、頼めば代行される、
  という両立へ意味が変わる)。

## 実装順序

1. E-1 src ops(N6 着地後、src 所有エージェント)
2. E-2〜E-4 web(IA v3 波A/B の後 — 同ファイルのため直列。波C i18n より前が
  望ましい: 新規文字列を辞書化対象に含めるため波C の直前に)
3. e2e: 分割/削除/undo/ショートカット/右クリックメニューのテスト追加

## 受け入れ基準

1. マウスだけ・キーボードだけのどちらでも、分割→移動→削除→undo の一連が
   迷いなくできる。
2. 右クリックに「その場でできること」が過不足なく出る(メニュー限定機能なし)。
3. ズームして 1F 単位の調整ができ、全体表示に一発で戻れる。
4. どの破壊的操作も Cmd+Z 一発で戻る。
5. 既存 e2e 全 green+新規操作の e2e が追加されている。
